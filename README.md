## üöÄ Quick Start

### Prerequisites

- Node.js >= 16.0.0
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/somritdasgupta/queFork.git

# Navigate to project directory
cd queFork

# Install dependencies
npm install --force

# Start the development server
npm run dev
```
